# Vite & Fidely UI Template

A starter template for building modern web applications with **Vite** and **Fidely UI** 

## Technologies Used

- Vite
- Fidely UI - latest
- Panda CSS
- TypeScript
- next-themes

### Getting Started

To clone the project, run the following command:

```bash
git clone https://github.com/fidely-ui/vite-ts-template.git
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
